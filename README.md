rover
=====

a cmd line util for [ranger](http://github.com/DHowett/ranger)


```
Usage of ./rover:
  -l	list files in zip
  -o string
    	the output filename
  -r string
    	the remote filename to download
  -t int
    	timeout, in seconds (default 5)
  -u string
    	the url you wish to download from
  -v	verbose
```
